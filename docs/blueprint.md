# **App Name**: TimeSnap

## Core Features:

- Real-time Attendance Tracker: User can instantly punch in and out with a large, dedicated button, with safeguards against invalid consecutive actions to ensure data integrity.
- Work Overview Dashboard: A prominent home screen displaying the current time, working status, today's accumulated hours, and calculated earnings, offering a quick and actionable glance at progress.
- Detailed History Log: A comprehensive, chronologically sorted list of all past work sessions, detailing check-in/out times, total duration, and calculated salary for each entry.
- Flexible Session Editing: Ability to easily modify details of any past work session, including check-in/out timestamps and an optional note, with automatic recalculation of hours and pay.
- Personalized Settings Panel: An interface to configure user-specific preferences such as hourly rate, currency symbol, and theme (e.g., dark mode toggle).
- Local Data Storage: All attendance records and settings are stored exclusively on the user's device, ensuring complete privacy, robust performance, and full offline functionality without needing a server.

## Style Guidelines:

- The primary light mode palette embodies a sense of calm professionalism and efficiency. The main interactive color is a subtle yet clear blue (#3872CC), complemented by a nearly white, soft blue background (#EEF3F9) to promote readability. An accent in cool cyan (#1F8EB3) provides a contrasting highlight. The application will also include a thoughtfully designed dark mode based on this palette.
- Using a single typeface for clarity and consistency, 'Inter' (sans-serif) is chosen for both headlines and body text, offering a modern, neutral, and highly readable aesthetic that aligns with the minimalist and efficient design philosophy.
- Leverage a set of clean, modern, and universally recognizable outline icons to ensure intuitive navigation and clear visual communication across all features, aligning with a professional and minimalist visual language.
- Adhere to a responsive grid system with generous padding and clear visual separation for elements. Primary calls-to-action, like check-in/out buttons, will be prominent and accessible. Information will be structured within rounded corner cards with subtle shadow effects for a contemporary, clean look, ensuring content hierarchy and ease of scanning.
- Incorporate subtle, performant micro-animations and transitions to provide fluid user feedback for interactions such as button presses, screen transitions, and data updates. The goal is to enhance the perception of 'smoothness' and responsiveness without introducing visual clutter.
