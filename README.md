
# TimeSnap Pro - Hệ thống Chấm công & Quản lý Lương Cá nhân

TimeSnap Pro là một ứng dụng web hiện đại (PWA) được thiết kế đặc biệt để giúp người lao động theo dõi giờ làm việc, tính toán lương tăng ca (OT) và quản lý thu nhập một cách chính xác.

## ✨ Tính năng nổi bật

- **Chấm công thời gian thực**: Vào ca/Ra ca chỉ với một nút bấm.
- **Tính toán OT thông minh**: 
  - Tự động nhận diện Ngày thường, Chủ nhật, Ngày lễ.
  - Quy tắc OT linh hoạt: Tự động tính x1.5 sau 8h làm việc.
- **Khấu trừ giờ nghỉ**: Tùy chỉnh số giờ nghỉ linh hoạt trong Cài đặt (để trống nếu không trừ).
- **Quản lý lịch sử**: Xem lại nhật ký làm việc theo từng tháng, hỗ trợ lọc Tháng/Năm.
- **Nhập/Xuất dữ liệu**: Hỗ trợ xuất báo cáo ra file CSV và nhập lại dữ liệu cực kỳ dễ dàng.
- **Giao diện PWA**: Cài đặt trực tiếp lên màn hình điện thoại như một ứng dụng Native.

## 🛠 Công nghệ sử dụng

- **Frontend**: Next.js 15, React, Tailwind CSS, ShadCN UI.
- **Backend**: Firebase Authentication, Firestore.

## 🚀 Hướng dẫn "Tải" ứng dụng (Cài đặt PWA)

Vì đây là ứng dụng Web App hiện đại, bạn không cần file .apk. Hãy làm theo cách sau:

1. **Trên Android (Chrome)**: Mở link ứng dụng -> Nhấn 3 chấm góc trên bên phải -> Chọn **"Cài đặt ứng dụng"** hoặc **"Thêm vào màn hình chính"**.
2. **Trên iPhone (Safari)**: Mở link ứng dụng -> Nhấn biểu tượng **Chia sẻ** (hình ô vuông có mũi tên lên) -> Chọn **"Thêm vào màn hình chính"**.
